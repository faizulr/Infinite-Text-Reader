## Description

Infinite Text Reader is a lightweight, browser-based tool designed to make reading text files easier and more enjoyable. Simply upload a .txt file and scroll through your content seamlessly without any interruptions.

## How to Use

1. Open `index.html` in your web browser
2. Click "Choose Text File" button or drag and drop a .txt file
3. Your text will be displayed instantly
4. Use the font size slider to adjust readability
5. Scroll naturally through your document
6. Click "Load New File" to read another document
