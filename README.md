## How to Use

1. Visit the [live demo](https://faizulr.github.io/Infinite-Text-Reader/)
2. Click "Upload" button or the "Choose File" button in the center
3. Select a .txt, .pdf, or .md file from your device
4. Your text will be displayed instantly (PDF files may take a moment to process)
5. Use the controls in the header to:
   - Adjust font size with + and - buttons
   - Toggle dark mode with the sun/moon icon
   - Enable auto-scroll with the play button
   - Adjust scroll speed (when auto-scroll is active)
6. Scroll naturally through your document or let auto-scroll do the work
7. Click "Upload" again to load a different file
